# ServerClientCalculator
Its a Server Client Calculator,where all the calculations are done at server Side.

Steps to use :
1. compile the server file using the command:
	gcc server.c -o server
2. compile the client file using the command:
	gcc client.c -o client
3. Now, run the client file using the command:
	./client
4. when it will ask for the operation use:
	a + b ( white space between operator and operand is necessary.
5. Can be used for operations like + , - , * and /.
